Redis time 命令

用于返回当前服务器时间。可用版本 >= 2.6.0

语法

time

返回值

一个包含两个字符串的列表： 第一个字符串是当前时间(以 UNIX 时间戳格式表示)，而第二个字符串是当前这一秒钟已经逝去的微秒数。

实例

Redis lastsave 命令

返回最近一次 Redis 成功将数据保存到磁盘上的时间，以 UNIX 时间戳格式表示。可用版本 >= 1.0.0

语法

lastsave

返回值

一个 UNIX 时间戳。
